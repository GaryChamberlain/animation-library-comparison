    Notes

    CSS let's you animate DOM and SVG content by statically defining named keyframes sequences. I chose standard DOM elements but that was just a design choice.

    There aren't a lot of named eases, but the cubic-bezier function can implement most of them. More advanced eases like bounces and elastics are not possible with a single bezier, though.

    The animation is occasionally jerky on my computer, especially when the animations first starts.
