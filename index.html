    • Comparison of Animation Libraries


    There is a plethora of ways to create animations on the web. Some tools use static definitions while others are coded with JavaScript. The various JavaScript libraries take different approaches and have varying sets of features. How does one determine which to use? I figured it would be a great exercise to build the exact same animation using each of many of the common tools and show how it's done.

    I wanted a simple animation so that the examples are easy to read and understand. However, I didn't want it to be so tedious as to not be interesting. It glides a ball back and forth on the X axis, using SIN wave easing, over two seconds. Simultaneously, the ball is in a container that is steadily rotating clock-wise at one revolution every eight seconds. This emits a pleasing, albeit simple, spiral pattern.
